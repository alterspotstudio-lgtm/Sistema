// READINESS SCORE — Airtable Formula
// Cuenta docs VALIDADOS + N/A / Total documentos aplicables × 100

ROUND(
  (COUNTIF({Doc_Escritura}, "Validado") +
   COUNTIF({Doc_Predial}, "Validado") +
   /* ... todos los campos de documentos ... */
   COUNTIF({Doc_INE_Vendedor}, "N/A")) /
  COUNTA({Docs_Aplicables}) * 100,
  0
)

// Cuando Readiness > 85 → Estado cambia a "Listo para cierre"

Automatizaciones Zapier
5 Zaps críticos que eliminan trabajo manual y previenen pérdidas
1
LEAD ENTRA AL SISTEMA
Formulario / Meta Ads / WhatsApp → Airtable + Mensaje
⚡
TRIGGER · New Lead from Meta Lead Ads / Typeform Se dispara cuando llega un nuevo lead del formulario o anuncio
+
ACTION 1 · Airtable — Create Record Crea registro en tabla "OPERACIONES NERI" · Estado = "Lead nuevo" · Score = 10 · Fecha entrada = NOW()
+
ACTION 2 · WhatsApp (Twilio / WATI) — Send Message Envía mensaje de bienvenida personalizado al número del lead · Template aprobado
+
ACTION 3 · Airtable — Update Record Estado → "Contactado" · Score +15 · Último contacto = NOW()
2
APARTADO REALIZADO → VIP
Activa expediente completo automáticamente
⚡
TRIGGER · Airtable — Record Updated Cuando campo "Apartado" cambia a TRUE en tabla OPERACIONES NERI
+
ACTION 1 · Airtable — Update Record Estado → "Cliente activo" · Prioridad → "VIP 🔥" · Score +25 · Fecha apartado = NOW()
+
ACTION 2 · WhatsApp — Send Checklist Envía mensaje de apartado + lista de documentos requeridos al cliente
+
ACTION 3 · Email / Slack — Notify Asesor Alerta al asesor responsable: "⚡ Cliente VIP — Inicia expediente AHORA"
3
INACTIVIDAD 48H → ALERTA RIESGO
Previene pérdida de clientes por descuido
⚡
TRIGGER · Zapier Schedule — Every Hour Corre cada hora para detectar inactividad
?
FILTER · Airtable Search Records Busca registros donde: {Días_Sin_Contacto} ≥ 2 AND {Estado} ≠ "Cerrado" AND {Estado} ≠ "Perdido"
+
ACTION 1 · Airtable — Update Record Prioridad → "Alto" (o marca como Riesgo en campo personalizado)
+
ACTION 2 · Email / Push — Alert Asesor "⚠️ [Nombre] sin contacto 48h · Actúa ahora antes de perder esta operación"
4
EXPEDIENTE >85% → LISTO PARA CIERRE
Activa alerta de cierre inmediato
⚡
TRIGGER · Airtable — Record Updated Cuando campo "% Avance Expediente" cambia y nuevo valor ≥ 85
+
ACTION 1 · Airtable — Update Record Estado → "Listo para cierre 🟢" · Score = 100
+
ACTION 2 · WhatsApp + Email — Cierre Alert Al asesor: "🟢 Expediente listo. Agenda firma. Comisión en juego: $XX,XXX"
5
CIERRE → COMISIÓN FINAL
Calcula y registra comisión · Actualiza dashboard
⚡
TRIGGER · Airtable — Record Updated Cuando {Estado} cambia a "Cerrado ✅"
+
ACTION 1 · Airtable — Update Record Comisión Confirmada = {Comisión Total} · Fecha Cierre = NOW()
+
ACTION 2 · Google Sheets (opcional) Agrega fila en hoja "Cierres Históricos" para reportes de dirección
+
ACTION 3 · Slack / Email — Celebración "🎉 ¡CIERRE! [Nombre] · Propiedad: $X · Comisión total: $X · Asesor: [Nombre]"
